## Step-by-Step (Takes ~5 minutes)

### Step 1: Create Repository on GitHub (1 minute)

1. Go to **[github.com](https://github.com)** → Sign in as `neubotix`
2. Click **"+"** (top-right) → **"New repository"**
3. Fill in:
   - **Repository name**: `wearable-optimizer-demo`
   - **Description**: "Interactive 3D wearable comfort optimizer"
   - **Visibility**: ✓ **PUBLIC**
   - **Initialize with README**: ✗ (uncheck)
4. Click **"Create repository"**

### Step 2: Upload Files via Command Line (2 minutes)

Open Terminal/PowerShell and run:

```bash
cd /path/to/wearable-optimizer-demo
git init
git add .
git commit -m "Initial commit: Wearable optimizer demo"
git remote add origin https://github.com/neubotix/wearable-optimizer-demo.git
git branch -M main
git push -u origin main
```

That's it! GitHub now has your files.

### Step 3: Enable GitHub Pages (1 minute)

1. Go to your repo: `https://github.com/neubotix/wearable-optimizer-demo`
2. Click **Settings** (top menu)
3. Left sidebar → Click **"Pages"**
4. Under "Build and deployment":
   - **Source**: Select **main** branch
   - Click **Save**
5. Wait 1-2 minutes ⏳

GitHub will show a message:
> "Your site is live at: https://neubotix.github.io/wearable-optimizer-demo/"
